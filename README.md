## Getting Started

### Prerequisites
- Node.js and npm installed

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/synsoft-global/REACT_NEXTJS_Movies_List.git

2. Navigate to the project directory:
   cd your-project

3. Install dependencies:
   npm install
